About this Project

Reto Taric, a web application written in Python under Django.
This application was developed by Guillermo Jimenez.

Taric_Books is a django application to provide a search engine of books. It allows searching books filtering by ISBN, title, author, publisher or subject.

The result will be displayed as a lists and you will be able to see book details by clicking on the book ISBN. Book details are recovered from ISBNdb.com and book covers are recovered from Google Books.
